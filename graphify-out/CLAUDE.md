# Instrucciones de la Sesión

- **Mapa de Arquitectura:** El proyecto cuenta con un grafo de conocimiento generado en la carpeta `graphify-out`. 
- **Regla de Oro:** Utiliza SIEMPRE este mapa JSON como tu referencia principal para entender la arquitectura, conexiones, dependencias o flujos de archivos del proyecto.
- **Eficiencia de Tokens:** No abras archivos de código completos ni uses herramientas de lectura a ciegas a menos que sea estrictamente necesario para realizar una edición.
